# FrontEndMap
以**思维导图**的形式梳理前端知识体系

此项目**每日更新**，会提供思维导图源文件及图片，有必要的地方会提供代码或者开issue，具体的会在对应文档中说明。

你可以用此项目来迅速巩固自己的知识体系，也可以由此步入前端的世界，希望可以对你有所帮助，个人精力和经验有限，如有错误之处，忘指出。
